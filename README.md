# SVHZ-Planetary-Habitability-Index-Catalog
Hello, my name is Srivathsan H.N. I have created a cataloge named SVHZ catalog. SVHZ catalog is a catalog of habitable exoplanet candidates derived from public astronomical datasets which provides habitability index scores, rating the planet's habitability zone and on how perfect it is. This helps us to find suitable candidates for life.

 ## Description
SVHZ (Stellar Habitable Zone created by Srivathsan H.N) is a catalog of potentially habitable
exoplanets identified using an python program

The catalog analyses known exoplanet systems and determines whether
planets fall within the calculated habitable zone of their host star.

## Data Sources
This catalog was generated using publicly available astronomical data
from the NASA Exoplanet Archive.

All habitability calculations, filtering methods, scoring metrics and catalog construction were performed independently by Srivathsan H.N.

This catalog represents an independent derived dataset and is not an official product of NASA or any affiliated institution. The data is categorised as the following Star,Planet,Orbit_AU,Luminosity,HZ_inner,HZ_outer,HZ_center,Habitability_Score
